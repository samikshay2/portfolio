The Boat Race

Project: Scratch Game

Course: Computer Science / Software Engineering (CSE)

Duration: Approximately 1.5 weeks (August 29 - September 8, 2017)

Objective: Create an original game with varying levels of difficulty and user interaction

Type of Project: Pair Programming (Samiksha Yelthimar and Derek Xu)

Link to Project (Game): The Boat Race Game

Link to Project Documentation: The Boat Race Documentation


In the Scratch Project, we had to create an original game of our choice. This was a two person project, and I was co-lead in the project. My partner for this project and I, thought that a boat-themed game could be very interesting if we added obstacles of varying difficulties. We also wanted to add multiple levels to this game, and give the player the choice to decide what level they want to start on. I focused strongly on the design, and making this game engaging through the use of various obstacles, such as sharks and oil spills. The objective of our game was to pass each level as fast as you can. If you hit the borders, or an obstacle, you will be prompted to start over from the beginning. It was difficult making all of the sprites to work as intented, especially working in the new Scratch environment, but we figured it out through trying over and over again. We created a Project Design Notebook that is linked under the section for project documentation. I learned various skills, including programming samples for computer science, and creating a Project Design Notebook for engineering,

Space Invaders

Project: MIT App Inventor Game

Course: Computer Science / Software Engineering (CSE)

Duration: Approximately 2.5 weeks (September 19 - October 6, 2017)

Objective: Improve, enhance, and add more features to a tutorial game of intermediate-advanced difficulty level

Type of Project: Pair Programming (Samiksha Yelthimar and Alicia Chen)

Link to Project (Game): Space Invaders Game (.aia file)

Link to Project Documentation: Space Invaders Game Documentation


In the MIT APP Inventor Project, we had to Improve, enhance, and add more features to a tutorial game of intermediate-advanced difficulty level. This was a two person project, and I was co-lead in the project. My partner for this project and I, thought that we could improve the Space Invaders game we were assigned by making the obstacles the rocket needs to dodge fall from the top of the screen, reduce the score if the rocket touches the obstacles, make the rocket shoot the obstacles, make the rocket shoot multiple bullets instead of one, and add new sprites. I focused strongly only the design aspect once again. It was difficult making all of the sprites to work as intented, especially working in the new MIT App Inventor environment, but we figured it out through trying over and over again. We created a Project Design Notebook that is linked under the section for project documentation. I learned various skills, including programming samples for computer science, and creating a Project Design Notebook for engineering,
